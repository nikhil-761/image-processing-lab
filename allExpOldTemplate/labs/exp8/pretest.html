Run-Length Encoding for Image Compression

Pre Test

1. Run-Length Encoding (RLE) is a form of which compression?

a) lossy
b) fractal coding
c) lossless
d) predictive

2. In RLE, a sequence of identical pixels is encoded as:

a) The pixel value and its count
b) Only the pixel value
c) The pixel and the next different pixel
d) The difference between successive pixels

3. RLE is most efficient for images that contain:

a) Many different colors mixed together
b) Complex photographic textures
c) Long runs of identical color (like simple graphics)
d) Random noise patterns

4. If an image has very few repeating pixels, RLE will:

a) Compress significantly
b) Leave the image unchanged
c) Possibly increase the file size (expand)
d) Produce lossy output

5. To decompress RLE data, the algorithm:

a) Repeats each pixel value according to its run-length
b) Applies the inverse DCT
c) Consults an adaptive dictionary
d) Performs wavelet reconstruction
