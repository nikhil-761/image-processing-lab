Sample and Quantization

Post Test

1. What is the primary consequence of reducing the sampling rate in an image?

a) Reduced file size
b) Increased bit depth
c) Loss of image details and resolution
d) Improved spatial resolution

2. Which of the following best describes quantization in digital image processing?

a) Selecting points from a continuous image
b) Increasing the spatial resolution
c) Mapping a continuous range of values into discrete levels
d) Filtering out noise from an image

3. According to the Nyquist theorem, the sampling frequency should be:

a) Equal to the image’s highest frequency
b) Half the image’s highest frequency
c) Double the image’s highest frequency
d) Irrelevant to the image frequency

4. Bit depth in an image defines:

a) The number of pixels per inch
b) The number of sampling points in an image
c) The image file format
d) The number of bits used per pixel to represent intensity

5. Why is image quantization important in compression algorithms like JPEG?

a) It increases the number of colors in the image
b) It reduces the amount of data by simplifying intensity levels
c) It sharpens the image
d) It improves edge detection
