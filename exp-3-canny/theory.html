Canny Edge Detection

Introduction

Image segmentation is the division of an image into regions or categories, which correspond to different objects or parts of objects. Every pixel in an image is allocated to one of these categories.

A good segmentation is typically one in which:

  • Pixels in the same category have similar grey-scale or multivariate values and form a connected region.
  • Neighbouring pixels belonging to different categories have dissimilar values.

Edge Based Image Segmentation

Edge-based segmentation is one of the most popular implementations of segmentation in image processing. It focuses on identifying the edges of different objects in an image. This is a crucial step because edges contain significant information about object boundaries and features.

Edge detection helps remove unwanted and unnecessary information from an image, reducing its complexity and making analysis easier. Algorithms used in edge-based segmentation identify edges based on differences in texture, contrast, grey level, colour, saturation, and other image properties.

Gradient

In image processing, a gradient refers to the rate of change of intensity in an image. Mathematically, the gradient is a vector field that points in the direction of the greatest rate of increase of intensity, while its magnitude represents the rate of change in that direction.

For edge detection, gradients are important because edges generally occur where there is a significant change in pixel intensity.

Steps in Edge Detection

  1. Filtering: Filtering is used to improve edge detector performance in the presence of noise. However, there is a trade-off between noise reduction and edge strength.
  2. Enhancement: Enhancement emphasizes pixels where there is a significant change in local intensity values, usually by computing gradient magnitude.
  3. Detection: Detection identifies pixels that actually belong to edges, often using thresholding techniques.
  4. Localization: Localization estimates the exact edge position and orientation with high precision.

Canny Edge Detector

The Canny edge detector is the first derivative of a Gaussian and closely approximates the operator that optimizes the product of signal-to-noise ratio and localization. The Canny edge detection algorithm begins by smoothing the image using a Gaussian filter.

Let I[i,j] represent the original image. The smoothed image is obtained as:

S[i,j] = S[i,j] = G[i,j;σ] * I[i,j]

where σ is the standard deviation of the Gaussian distribution and controls the amount of smoothing applied to the image.

The gradient of the smoothed image is then computed using finite difference approximations to obtain the horizontal and vertical derivatives:

P[i,j] ≈ ( S[i,j+1] - S[i,j] + S[i+1,j+1] - S[i+1,j] )/2

Q[i,j] ≈ ( S[i,j] - S[i+1,j] + S[i,j+1] - S[i+1,j+1] )/2

The finite differences are averaged over a 2×2 neighbourhood so that the horizontal and vertical derivatives are computed at the same location.

The gradient magnitude and orientation are then calculated using:

M[i,j] = P[i,j] 2 + Q[i,j] 2

θ[i,j] = arctan ( Q[i,j] , P[i,j] )

The arctangent function generates an angle over the entire range of possible directions. The Canny detector then performs non-maximum suppression and hysteresis thresholding to produce thin and accurate edges.

Comparison of Edge Detectors

Edge Detector Method Advantages Limitations
Roberts Gradient Based
  1. Easy and simple computation.
  2. Edges detected with orientation.
  1. Highly sensitive to noise.
  2. Less accurate edge detection.
  3. Less reliable.
Sobel Gradient Based
  1. Easy implementation.
  2. Provides edge orientation.
  1. Sensitive to noise.
  2. Moderate accuracy.
  3. May blur fine edges.
Prewitt Gradient Based
  1. Simple computation.
  2. Efficient for basic edge detection.
  1. Noise sensitive.
  2. Less accurate than Sobel.
  3. Weak edge localization.
Canny Gaussian Based
  1. Improved signal-to-noise ratio.
  2. Suitable for noisy images.
  3. Accurate edge localization.
  4. Produces thin and continuous edges.
  1. Computationally expensive.
  2. More complex implementation.
  3. Threshold selection affects performance.

Conclusion

Edge detection is one of the most important techniques in image segmentation and image analysis. Gradient-based operators such as Roberts, Sobel, and Prewitt detect edges using first-order derivatives, while the Laplacian uses second-order derivatives. Among these methods, the Canny edge detector is widely considered one of the most effective because it combines Gaussian smoothing, gradient computation, non-maximum suppression, and hysteresis thresholding to achieve accurate and reliable edge detection.

Proper edge detection helps reduce image complexity, improve object recognition, and enhance the performance of various image processing and computer vision applications, including medical imaging, remote sensing, industrial inspection, and autonomous systems.
